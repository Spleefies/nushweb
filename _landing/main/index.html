NUSHWeb

NUS High's student-made school portal
By Students, For Students

Welcome to NUSHWeb, a web portal created by the students of NUS High School! This website is not affiliated with the school in any way - it's just a side project of the creators.


Full Integration

NUSHWeb utilises the school-issued email/Microsoft accounts to enable users to login, removing the need for students to re-register.

Students' names and Microsoft IDs are only stored on the user's device. Personalisation features are stored on the backend, which stores only users' student ID (a 7-digit number created by the school and available through Microsoft login).

Convenience and Accessibility

The website can be installed as a web app on the user's mobile device, enabling more features that you would expect to see in a native app.

The app can then be accessed via an icon on the user's home screen, for quick and convenient access.

Familiarity

The home page of the portal shares some common features with the MOE Student Learning Space portal, which most students would already be familiar with. In addition, some common features found on other websites are also present.

This allows first-time users to quickly familiarise themselves with the features of the application.
